What is NodeJS ?

Node.js is an open-source, cross-platform, server-side JavaScript runtime environment. It allows developers to use JavaScript on the server-side to build scalable, high-performance web applications. Node.js was created by Ryan Dahl in 2009 and is built on top of the V8 JavaScript engine from Google Chrome.

What is GitHub.

GitHub is a web-based platform that provides a central repository for storing and managing version-controlled source code. It is built on top of the Git version control system, which allows developers to collaborate on code changes and track revisions over time.
